# Introduction to DNAtranslate

the DNAtranslate provides utility for common tasks in analyzing DNA
within R. To install this package from GitHub, use

``` r

# install.packages("remotes")
# install.packages("devtools")
devtools::install_github("ADC-405-S26/DNAtranslate")
```

To load the package use the following code

``` r


library(DNAtranslate)
```

The package provides three functions \* ‘readSequence’ \* ‘translateDNA’
\* ‘reverseSequence’
