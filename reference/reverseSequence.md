# Reverses a specified sequence

Reverses a specified sequence

## Usage

``` r
reverseSequence(df, name)
```

## Arguments

- df:

  a dataframe with exactly 2 columns, typically generated by the
  readSequence function in this package

- name:

  the name of the sequence that needs to be reversed

## Value

the "same" dataframe but with the specified dataframe reversed

## Examples

``` r
sequences <- reverseSequence(exampleSequences, "HisD TA1538 Mutant Genotype")
```
