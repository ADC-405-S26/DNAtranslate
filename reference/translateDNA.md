# Translate a DNA sequence to a protein sequence

Translate a DNA sequence to a protein sequence

## Usage

``` r
translateDNA(df, offset = 0)
```

## Arguments

- df:

  a dataframe with exactly 2 columns, typically generated by the
  readSequence function in this package

- offset:

  a number 0,1,2 that determines the reading frame offset from the start
  of the

## Value

a dataframe with 3 columns, name, DNA sequence, protein sequence

## Examples

``` r
protein_df <- translateDNA(exampleSequences)
```
